# 영남수산 고객용 웹서비스 PRD

## 1. 프로젝트 개요

프로젝트명:
yn-customer

서비스명:
영남수산 고객용 웹서비스


## 2. 프로젝트 목적

카카오톡 알림톡을 받은 고객이 방문하는 모바일 중심 고객 서비스 페이지 제작.

단순 배송조회 기능만 제공하는 것이 아니라,
영남수산 고객이 구매 이후 필요한 정보를 쉽게 확인할 수 있는 통합 고객 서비스로 확장한다.

주요 목적:

- 배송 상태 확인 편의성 제공
- 수산물 보관 방법 안내
- 맛있게 먹는 방법 안내
- 재주문 및 문의 연결
- 향후 고객 서비스 확장 기반 구축


## 3. 서비스 대상

주 사용자:

- 과메기 구매 고객
- 문어 구매 고객
- 수산물 구매 고객
- 모바일 사용자가 대부분

특징:

- 카카오톡 링크를 통해 접속
- 앱 설치 없음
- 스마트폰 사용자가 대부분
- 40~70대 고객도 쉽게 사용할 수 있어야 함


# 4. 핵심 기능


## 4-1. 배송조회

페이지:

/tracking


목표:

고객이 운송장번호를 직접 입력하지 않고
알림톡 링크 클릭만으로 배송 상태 확인


사용 흐름:

yn-order-manager
↓
알리고 알림톡 발송
↓
고객 배송조회 버튼 클릭
↓
yn-customer /tracking 이동
↓
안전한 인증 토큰 확인
↓
배송정보 요청
↓
배송 상태 표시


화면 구성:

영남수산 배송조회

상품명:
과메기 세트

택배사:
CJ대한통운

운송장번호:
자동 표시

현재 배송상태:

상품 준비
↓
집하 완료
↓
배송 이동 중
↓
배송 완료


표시 정보:

- 상품명
- 택배사
- 배송상태
- 현재 위치
- 배송 진행 단계



## 4-2. 수산물 보관방법 안내

페이지:

/guide/storage


목적:

상품:

- 과메기
- 문어
- 반건조 생선
- 전복
- 기타 수산물


상품별 제공 정보:

- 보관 방법
- 냉장/냉동 여부
- 권장 보관 온도
- 보관 가능 기간
- 해동 방법
- 주의사항
- 이미지 영역



## 4-3. 맛있게 먹는 법 안내
